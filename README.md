# 🎯 Jogo do Número Secreto

Projeto desenvolvido durante os estudos de **JavaScript** na Alura.

O objetivo do jogo é descobrir qual é o número secreto gerado pelo sistema.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎮 Como funciona o jogo

1. O sistema gera um número secreto entre **1 e 10**.
2. O jogador tenta adivinhar o número.
3. O sistema informa se o número é **maior ou menor**.
4. O jogo continua até o jogador acertar.
